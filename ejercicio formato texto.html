Glosario de Aplicaciones Web

Ejemplos de servidores web

APACHE:el más popular y extendido, es gratuito y de código abierto, tiene una desfasada arquitectura comparándolo con otros servidores web. NGINX:gratuito y de código abierto con versión de pago, tiene un alto rendimiento, usa un proxy inverso para proteger la identidad de los servidores y mejorar la seguridad de la información que tienen. Configuración sencilla y personalizable, consume pocos recursos. MICROSOFT IIS:está creado íntegramente para los sistemas operativos de Windows por lo que es compatible con todas las multinacionales estadounidenses, su defecto es la integración de lenguajes y tecnologías. JAVA SYSTEM WEB SERVER:gestiona grandes cargas de trabajo de Java, Python o Ruby on Rails, está pensado como herramienta para programadores

APACHE: The most popular and widespread, it is free and open source, though its architecture is outdated compared to other web servers. NGINX:Free and open source with a paid version, it offers high performance and uses a reverse proxy to protect server identities and enhance the security of the information they hold. It has simple, customizable configuration and consumes few resources. MICROSOFT IIS:Entirely built for Windows operating systems, making it compatible with all American multinational companies. Its drawback is the integration of languages and technologies. JAVA SYSTEM WEB SERVER: Manages large workloads of Java, Python, or Ruby on Rails. It is designed as a tool for developers.

LAMP y XAMPP:

LAMP: Es una abreviatura de las palabras Linux, Apache, MySQL y PHP. Esas tecnologías se utilizan para el desarrollo de aplicaciones web. Linux es el sistema operativo base, Apache es un software que te permite establecer una conexión entre servidor y navegadores web mientras se envían archivos entre ellos, MySQL es un sistema de gestión de bases de datos de código abierto y PHP es un lenguaje de programación que se utiliza para desarrollar las aplicaciones web. XAMPP: está diseñado para ser fácil de instalar, permite crear y probar antes de que salga a la luz, tiene una interfaz que te permite gestionar y configurar fácilmente los diferentes componentes de stack, es compatible con windows, MacOS y Linux, y te permite desarrollar una gran variedad de aplicaciones web.

LAMP:It is an acronym for Linux, Apache, MySQL, and PHP. These technologies are used for web application development. Linux is the base operating system, Apache is software that allows you to establish a connection between the server and web browsers while files are being exchanged, MySQL is an open-source database management system, and PHP is a programming language used to develop web applications. XAMPP:It is designed to be easy to install, allowing you to create and test before going live. It has an interface that lets you easily manage and configure the different components of the stack. It is compatible with Windows, macOS, and Linux, and allows you to develop a wide variety of web applications.

HTML

Se utiliza para estructurar el contenido de un sitio web para definir la apariencia del sitio web (como encabezados, párrafos, enlaces, etc). HTML se utiliza junto con otros lenguajes y tecnologías como CSS (Cascading Style Sheets) para el diseño y presentación, y JavaScript para interactividad.

It is used to structure the content of a website to define the appearance of the site (such as headers, paragraphs, links, etc.). HTML is used along with other languages and technologies like CSS (Cascading Style Sheets) for design and presentation, and JavaScript for interactivity
