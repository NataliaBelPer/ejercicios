Glosario de términos en relación con HTML

HTML
Es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.
CSS
Ordena las instrucciones referentes a la apariencia de un sitio y presentar los contenidos de una página de forma atractiva
JavaScript
Crea contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás
Python
Lenguaje de programación ampliamente utilizado en las aplicaciones web, el desarrollo de software, la ciencia de datos y el machine learning
